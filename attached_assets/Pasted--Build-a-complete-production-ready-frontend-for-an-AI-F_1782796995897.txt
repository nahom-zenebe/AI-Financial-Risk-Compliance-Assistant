
Build a complete production-ready frontend for an AI Financial Risk & Compliance Assistant.

TECH STACK:
- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Zustand (state management)
- JWT authentication (stored in localStorage)
- Lucide React icons
- Axios for API calls
- React Hook Form (optional for forms)

BACKEND:
- FastAPI backend running separately
- Base URL: http://localhost:8000

------------------------------------------------------------
PROJECT GOAL:
------------------------------------------------------------
Build a professional AI dashboard frontend where users can:

1. Login/Register using JWT
2. Upload financial documents (PDF, DOCX, CSV)
3. Chat with AI (RAG + Gemini backend)
4. View compliance reports
5. Analyze transaction risk
6. View dashboard analytics

------------------------------------------------------------
AUTH SYSTEM (JWT):
------------------------------------------------------------

- Login page sends request to:
  POST /auth/login

- Response:
  {
    access_token: string,
    user: {
      id,
      email,
      role
    }
  }

- Store token in Zustand + localStorage

- Add Axios interceptor:
  - Attach JWT token to every request

- Protected routes:
  - redirect to /login if no token
